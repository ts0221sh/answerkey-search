使い方

1. 変換キーを入れる

リスト1(変換元)とリスト2(変換先)を1文字ずつ対応させて入力します。同じ長さである必要があります。

リスト1: あいうえおかきくけこ
リスト2: abcdefghij
→ 「あうけ」は1,3,9番目なので「ace」に変換

リスト1に同じ文字が複数登場する場合、その文字は複数の変換候補を持ちます。全パターンで変換を試して、出力辞書にヒットしたものすべてが結果になります。

ダミー文字「/」「#」: 見やすさのために区切りや空白として使えます。リスト1かリスト2のどちらかにダミー文字がある位置は、マップから除外されます(その対応は試されません)。両方のリスト長は同じになるよう揃えてください。

2. 辞書を選ぶ

入力辞書: 変換元の単語が含まれる辞書(複数選択でマージ)。
出力辞書: 変換結果が存在するか照合する辞書(複数選択でマージ)。

辞書はGAS連携・内蔵辞書・ファイル・手入力の4通りで読み込めます。

3. オプション

不変文字: 入力単語の中で「リスト1に含まれない文字」を最大何個まで許容するか選択します。

  • なし: 全文字がリスト1にある単語のみ変換
  • 1つまで: リスト1にない文字を1個まで許容(その文字は不変)
  • 2つまで: リスト1にない文字を2個まで許容(その文字は不変)

入力=出力 を除外: 変換後の単語が変換前と完全に一致するもの(全く変化しなかったもの)を結果から除外します。

アナグラム許容: 変換結果そのものが出力辞書になくても、変換結果を並び替えた単語が出力辞書にあればヒットさせます。結果テーブルでは [A] マーク、コピー時のコメントに ,アナグラム が付加されます。

清濁増減: 変換キーから清音/濁音/半濁音の派生ペアを自動追加します。

  • 清×清 → 濁化版を追加 (例: は→き ⇒ ば→ぎ)
  • 清×清 (両方ハ行) → 半濁化版も追加 (例: は→ひ ⇒ ば→び, ぱ→ぴ)
  • 濁×濁 → 清化版を追加。両方ハ行濁音なら半濁化版も追加
  • 半濁×半濁 (=両方ハ行) → 清化版・濁化版を追加
  • 濁×半濁 → 「点を外す」操作で両方清化版を追加 (例: ば→ぴ ⇒ は→ひ, が→ぴ ⇒ か→ひ)
  • 清×濁、清×半濁 はスキップ

大小増減: 変換キーから大小の派生ペアを自動追加します。対象は あいうえおつやゆよ ⇔ ぁぃぅぇぉっゃゅょ。

  • 両方が大文字 → 両方を小文字化したペアを追加
  • 両方が小文字 → 両方を大文字化したペアを追加
  • 大×小 の混在はスキップ

ずらしマッチ: 通常のヒット結果に対して、入力・出力の文字をシフトリスト上で動かしたバリエーションを試行。3パターンのモードから選択:

  • OFF: ずらしを試さない
  • 両方: 入力と出力を同じ k だけずらす(両方が辞書にあれば追加)
  • 片方: 入力のみずらし(出力固定) と 出力のみずらし(入力固定) を試す
  • 両方+片方: 上記すべてを試行
シフトリスト:
  • かな(46文字、ループ): あいう…をん
  • 濁音(15、ループ禁止): がぎ…でど
  • 濁音バ行(5、ループ禁止): ばび…べぼ
  • 半濁音(5、ループ禁止): ぱぴ…ぺぽ
  • アルファベット(26、ループ): a〜z
同リスト同士はリスト長-1までシフト試行(ループ許容なら+方向のみ、ループ禁止なら両方向)。派生(清濁/大小)由来のヒットはシフトの対象外。 結果テーブルでは [S+k](両方)、[Si+k](入力のみ)、[So+k](出力のみ) のマークで区別。

4. 結果の操作

文字数で昇順/降順ソート、最小/最大文字数でフィルタができます。コメント欄に文字を入れてから「フィルタ後をコピー」を押すと、フィルタ後の全行を 入力,出力,コメント 形式でクリップボードに保存します。

コメント末尾が (不変なし) (不変あり) (不変あり×2) の場合、行ごとに以下が自動で置換されます:

  • 不変文字を使わずに変換可能 → (不変なし)
  • 不変文字を1個使った行 → (不変あり)
  • 不変文字を2個使った行 → (不変あり×2)
  • アナグラム結果 → ,アナグラム が追加
  • 入力と出力の「位置ごとに違う文字の数」 → ,N文字違い が追加

5. デバッグログ・ハッシュ

うまくいかない場合は「デバッグログをコピー」で内容を共有してください。状態ハッシュを使えば、入力条件と辞書選択を完全再現できます。
